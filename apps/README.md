# Userspace Applications

GUI and desktop components live here as **userspace programs** - not kernel code.

## Planned Apps

| App | Path | Milestone |
|-----|------|-----------|
| Window Manager | `apps/wm/` | M7 |
| Desktop Shell | `apps/desktop/` | M7 |
| Terminal | `apps/terminal/` | v0.2+ |
| File Manager | `apps/files/` | v1.0 |
